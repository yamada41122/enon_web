⚙ GitHub 接続設定

「🚀 サイトに公開」ボタンで、編集内容を直接GitHubにコミットするための設定です。Tokenはこのブラウザ内にのみ保存され、外部には送信されません。

📘 Personal Access Tokenの作成手順

  1. Fine-grained Personal Access Token 発行ページを開く
  2. Token name: enon-admin など
  3. Expiration: 90日など任意
  4. Repository access: Only select repositories → enon_web を選択
  5. Permissions → Repository permissions で Contents を Read and write に設定
  6. Generate token → 表示された github_pat_... をコピーして上の欄に貼り付け
  7. 「設定を保存」→「接続テスト」で確認

🔒 セキュリティ: Tokenはお使いのブラウザのlocalStorageにのみ保存されます。他人と共有しているPCでは使用後に「Tokenを削除」を押してください。

使い方: 編集内容は自動で保存（ブラウザ内）されます。 🚀 サイトに公開 ボタンで直接GitHubにコミット → 数十秒〜数分で本番サイトに反映されます。 ダウンロード/アップロード方式でもOKです。
